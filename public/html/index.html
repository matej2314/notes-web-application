Twoje notatki:

Witam na stronie umożliwiającej zarządzanie własnymi notatkami.

Po zalogowaniu będziesz mógł:

  • stworzyć swój profil użytkownika,
  • dodawać notatki,
  • edytować notatki,
  • usuwać notatki,
  • zapisywać wybrane przez siebie notatki do pliku PDF.

Technologie wykorzystane przy tworzeniu strony:

  • HTML
  • CSS (Tailwind CSS)
  • JavaScript
  • Node.js

Załóż konto i sprawdź funkcjonowanie aplikacji samodzielnie!
